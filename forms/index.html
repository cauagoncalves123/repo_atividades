Formulário em HTML

Um formulário em HTML é uma maneira fácil de coletar informações de usuários. Ele é delimitado por um elemento <form> e pode conter vários elementos <input> e outros controles.

Como funciona o container form?

O elemento <form> é o "container" do formulário. Ele agrupa todos os elementos de entrada e define o método de envio de dados (GET ou POST).

Aqui está um exemplo simples:

Código:


    <form>
        <p>
            <label for="name">Nome:</label>
            <input type="text" id="name" name="name">
            <br>
        </p>
        <p>
            <label for="email">E-mail:</label>
            <input type="email" id="email" name="email">
            <br>
        </p>  
        <button type="submit">Enviar</button>
    </form>

Quais os tipos de input?

Existem vários tipos de entrada em HTML, cada um com um propósito diferente. Alguns dos tipos mais comuns são:

  • text: caixa de texto simples
  • email: caixa de texto para entrada de e-mail
  • password: caixa de texto para entrada de senha
  • number: caixa de texto para entrada de número
  • checkbox: caixa de seleção
  • radio: botões de opção
  • submit: botão de envio do formulário

Aqui está um exemplo de como usar alguns desses tipos:

Código:


    <form>
        <p>
        <label for="name">Nome:</label>
        <input type="text" id="name" name="name">
        </p>
        <p>
        <label for="email">E-mail:</label>
        <input type="email" id="email" name="email">
        </p>
        <p>
        <label for="pwd">Senha:</label>
        <input type="password" id="pwd" name="pwd">
        </p>
        <p>
        <input type="checkbox" id="newsletter" name="newsletter" value="yes">
        <label for="newsletter">Desejo receber newsletter</label>
        </p>
        <p>
        <label for="gender">Gênero:</label>
        <label for="male">Masculino</label>
        <input type="radio" id="male" name="gender" value="male">
        <label for="female">Feminino</label>
        <input type="radio" id="female" name="gender" value="female">
        </p>
        <p>
        <button type="submit">Enviar</button>
        </p>
    </form>

Como funciona o post em HTML?

O método POST é usado para enviar dados para o servidor. Quando um formulário com o método POST é submetido, os dados são enviados no corpo da solicitação HTTP e não são exibidos na URL. Isso torna o método POST mais seguro do que o método GET.

Aqui está um exemplo de como usar o método POST:

Código:


    <form action="http://example.com/submit" method="post">

        <input type="text" name="nome" value="Seu nome aqui">

        <input type="submit" value="Enviar">

    </form>
